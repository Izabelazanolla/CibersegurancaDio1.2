# CibersegurancaDio1.2

O projeto simula, de forma simples e inofensiva:

Um ransomware — apenas codifica arquivos de teste usando Base64 (não usa criptografia real).

Um keylogger — apenas gera um arquivo com texto fictício simulando teclas digitadas (não captura teclado de verdade).

Reflexão sobre defesa — inclui um pequeno material explicando boas práticas de proteção.

Essas simulações ajudam a entender:

Como um malware poderia se comportar.

Como proteger sistemas contra ameaças reais.

Como montar um ambiente seguro para testes.
